# 🚀 AI Skill Gap Analyzer

An intelligent full-stack web application that analyzes a student's career readiness and identifies skill gaps using AI-driven insights.

---

## 🌟 Overview

AI Skill Gap Analyzer helps students:

- 📄 Upload resumes
- 💻 Add coding statistics
- 📊 View readiness analysis
- 🔍 Explore tech domains
- 🎯 Identify missing skills
- 📈 Improve placement preparation strategy

This project is built with modern full-stack technologies and designed with an industry-level UI.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- Lucide Icons

### Backend
- Node.js
- Express.js
- MongoDB

## 🚀 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-skill-gap-analyzer.git
cd ai-skill-gap-analyzer
```

---

### 2️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
npm start
```

Runs on:  
👉 https://ai-skill-gap-analyzer-1-2p7n.onrender.com

---

### 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
node server.js
```
